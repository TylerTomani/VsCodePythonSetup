Setting up Visual Studio Code

1.

 First, you'll need to install the python intellisense extension. Go to the four square icon and type in

  • python

 Then choose the first one called "intellisense" and install it.

2.

 Check if visual studio is using the correct version of python.

 Make a new file in visual studio

  • command + N

or

  • File→new Text File

3.

 Save a file, name it whatever you want, just end it with .py

 Naming a file .py automatically makes it a python script.

4.

 The version of python will now show up at the bottom of the screen

5.

 If for some reason, you don't see the version at the bottom. Type

    command + shift + p

or go to,

    View→Command Palette

6.

 Python should appear, if not, just type in,

  • python interpreter

 Choose the latest one you installed 3.10

7.

 Last thing, This is important down the road, so you can have scripts work with other scripts.

Go to Settings by typing.

  • command + ,

or

    Code→Preferences→Settings

8. Last step

 In the search bar type,

  • executeinfiledir

 Make sure that box is checked

Extra Tip

 Change the color so your eyes don't hurt.

 Go to,

  • Preferences→Settings→workbench→Appearance→Color Theme

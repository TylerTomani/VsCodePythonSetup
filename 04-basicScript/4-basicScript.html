Basic Python Script

 The print function prints to the console.

 To run the script in the console, right click anywhere in the window and select Run Python file in Terminal

 Or, you can hit the play button in the top right

 Hit command + j to open and close the terminal

Basic Error

 If you come across this error, dont' worry, just type

    exit()

And run the script again. I used the short cut I made here which is

  • command + alt + \

Create a shortcut

 The best thing to do is to create a shortcut for running a script.

To create a short cut, go to

  • Code→Preferences→Keyboard shortcuts

type in,

  • run python

Then double click the keybinding column
